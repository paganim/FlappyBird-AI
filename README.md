# FlappyBird-AI
This is a Flappy Bird AI with NEAT

# How it works
Perform these steps:
```
Fork this repo
Clone this repo
cd flappybird-ai
```
On terminal run this command:
```
python main.py
```
If you want to change same setting you can open config-feedforward.txt file (fitness, population, etc....)
